# unsub

CLI tool to migrate from SaaS to self-hosted alternatives.

## Features

- Migrate data from various SaaS platforms to self-hosted solutions
- Support for Notion, and more
- Progress tracking with CLI progress bar
- Beautiful CLI output with chalk and ora

## Installation

```bash
# Clone the repository
git clone https://github.com/Moon7z/unsub.git
cd unsub

# Install dependencies
npm install

# Build the project
npm run build

# Install globally (optional)
npm link
```

## Usage

```bash
# Run the CLI
npm start

# Or use after npm link
unsub --help
```

## Development

```bash
# Watch mode
npm run dev

# Build
npm run build
```
